# ENTRENADOR EDUCATIVO (app online - by Javi Ortiz)
Juego interactivo que genera 100 preguntas tipo test sobre un temario cargado a partir de fotos que puedes subir. 
Dispone también de una DEMO, con preguntas relativas al curso escolar del alumno (todas las asignaturas).
Podrás generar nuevos Test por cada asignatura y tema que prefieras, darlos de baja si ya no los necesitas, ...
Contacta con el Administrador y obten tu PIN de acceso al juego (enviale tu Nombre y Curso)
