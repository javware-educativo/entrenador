⚙️ Configuración
🔑 API Key Anthropic (IA)
Necesitas una clave API de Anthropic para que la IA genere preguntas.
Si cambias la clave, simplemente pega la nueva y pulsa Guardar.
📋 PASOS PARA OBTENER TU CLAVE API
1️⃣ Ve a platform.claude.com/settings/keys
2️⃣ Inicia sesión con tu cuenta de Anthropic (o crea una)
3️⃣ Añade un método de pago en Billing y carga mínimo $5
4️⃣ Ve a API Keys -> pulsa "Create Key"
5️⃣ Ponle un nombre (ej: "Entrenador") y copia la clave completa
6️⃣ Pégala aquí abajo y pulsa Guardar
💰 Coste aproximado: 2-5 céntimos por tema generado (100 preguntas)
